Como renomear colunas de uma tabela (data.frame) com o pacote dplyr?

Dec 13, 2018 3 min read tutorial

O pacote dplyr

“dplyr é uma gramática para a manipulação de dados, oferecendo uma série de verbos consistentes que ajudam você a resolver os principais desafios de manipulação de dados e tabelas no R” (https://dplyr.tidyverse.org)

rename()

Neste tutorial iremos aprender a usar o verbo rename() (renomear).

A função renomear() modifica o nome de uma ou mais colunas de uma tabela de dados. Todas as colunas são mantidas.

Forma de uso

A forma de usar a função segue essa formula:

dados <- rename(dados, new_name = old_name)

Este comando irá modificar o nome da coluna “old_name” para “new_name”. Vamos ver como funciona na prática.

Exemplos

A tabela abaixo possui 5 colunas!

iris <- as_tibble(iris) # transformando em tibble (não é necessário)

Suponha que queremos modificar o nome da coluna “Species” e trocar por “SP”.

Comando sem dplyr

colnames(iris)[5] <- "sp"
iris

Comando com dplyr

iris <- rename(iris, SP = Species)
iris

Agora suponha que queremos modificar o nome das colunas “Sepal.Length” e “Sepal.Width para “sepL” e “sepW”.

iris <- rename(iris, sepL = Sepal.Length, sepW = Sepal.Width)

Note que não precisamos colocar o nome das colunas entre aspas, o que facilita muito o processo. Além disso, podemos renomear múltiplas colunas ao mesmo tempo sepando por vígula.
